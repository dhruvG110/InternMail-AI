# 📖 Overview

InternTrack AI is a modern AI-powered SaaS platform that helps students and professionals manage every stage of their job search from one dashboard.

Instead of maintaining spreadsheets, notes and multiple resumes, users can:

- Track applications
- Schedule interviews
- Generate AI career advice
- Analyze resumes
- Generate follow-up emails
- View analytics
- Manage multiple resume versions

# 🚀 Installation

```bash
git clone https://github.com/YOUR_USERNAME/InternTrack-AI.git

cd InternTrack-AI

npm install
```

---

## Environment Variables

```env
DATABASE_URL=

NEXTAUTH_SECRET=
NEXTAUTH_URL=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

GEMINI_API_KEY=
```
